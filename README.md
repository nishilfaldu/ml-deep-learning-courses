A personal archive of the machine-learning and deep-learning courses I worked
through when I was first getting into programming — the assignments, notebooks,
and from-scratch implementations, kept exactly as I built them.

> When I first started programming, I stumbled into machine learning and fell in
> love with the math behind it. I found it so fascinating that I started taking
> every famous ML and deep-learning course I could find, working through the
> assignments one notebook at a time. This repo is the record of that — from
> linear regression in Octave to building RNNs that generate jazz. I'm keeping it
> because I plan to go deeper into ML again from first principles, and because it
> captures something I care about: I like learning hard things for their own sake.

The last four courses together make up Andrew Ng's **Deep Learning
Specialization**; the first is his original **Stanford Machine Learning** course,
which is why it's written in Octave/MATLAB rather than Python.

## The assignments

**Machine Learning (Stanford)** — implemented from the ground up in Octave:
linear regression, logistic regression, multi-class classification and neural
networks, neural network learning (backprop), regularized regression and the
bias/variance tradeoff, support vector machines, K-means and PCA, and anomaly
detection with recommender systems.

**Neural Networks and Deep Learning** — Python basics with NumPy, logistic
regression as a single neuron, a one-hidden-layer planar classifier, a deep
neural network built layer by layer, and an image-classification application.

**Improving Deep Neural Networks** — weight initialization strategies,
regularization, gradient checking, optimization methods (momentum, Adam), and a
first TensorFlow tutorial.

**Convolutional Neural Networks** — convolution model fundamentals, a Keras
tutorial, ResNets, car detection for autonomous driving (YOLO), neural style
transfer, and face recognition.

**Sequence Models** — recurrent neural networks, a character-level language model
("Dinosaur Island"), jazz improvisation with an LSTM, Emojify, neural machine
translation with attention, and trigger-word detection from audio.
